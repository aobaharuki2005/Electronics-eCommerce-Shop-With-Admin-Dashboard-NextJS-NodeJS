# Về Repo này
Đây là clone của project mã nguồn mở [Kuzma02/Electronics-eCommerce-Shop-With-Admin-Dashboard-NextJS-NodeJS](https://github.com/Kuzma02/Electronics-eCommerce-Shop-With-Admin-Dashboard-NextJS-NodeJS).
Mục đích là phục vụ cho đồ án nghiên cứu học phần Project 1 tại Đại học Bách Khoa Hà Nội.

# Những file sản phẩm của đồ án nghiên cứu Project 1
1. File cấu hình Workflow quét và build bảo mật: `/.github/workflow/security-scan-and-build.yml`
2. File cấu hình SonarQube: `/sonar-project.properties`
3. File cấu hình Dependabot (quét và tạo PR cập nhật bản vá tự động): `/.github/dependabot.yml`
4. File cấu hình cho Docker (Dockerfile): `/Dockerfile`
5. File mẫu cho việc tải về và khởi chạy Docker Image: `/docker-compose.yaml`
